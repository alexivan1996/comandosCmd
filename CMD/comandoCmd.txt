cd+nombre = sirve para ingresar a una carpeta 
cd .. = sirve para retroceder una carpeta
dir = sirve para ver el contenido de  una carpeta 
tree = sirve para listar las capetas (arbol de carpetas)
cls = limpiar nuestro CMD
mkdir = sirve para crear una carpeta (directorio)
rd o mrdir = sirve para eliminar carpeta 
copy con = sirve para crear un  archivo y editarlo 
ctrl+z = sirve para  guardar el archivo
exit = salir del cmd
